## 代码使用教程

1.安装requirements里的依赖项

```
pip install -r requirements.txt
```

2.打开代码文件所在的命令行，安装playwright

```
playwright install
```

3.下载node.js

参考[Node.js安装及环境配置超详细教程【Windows系统】_windows 安装nodejs-CSDN博客](https://blog.csdn.net/Nicolecocol/article/details/136788200)，简单无坑，只要记得下载后加入环境变量即可

4.安装vue

打开代码文件所在命令行，输入

```
npm install
```

5.运行代码

打开代码文件所在命令行

```
python server.py
```

再打开一个命令行

```
npm run dev
```
